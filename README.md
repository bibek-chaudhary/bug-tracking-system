# Bug Tracking System

A full-stack Bug Tracking System built with:
- ASP.NET Core 8 Web API
- React
- SQL Server
- Entity Framework Core (Code-First)
- ASP.NET Core Identity + JWT
- Clean Architecture

## Project Structure

bug-tracking-system/
├── backend/ # .NET 8 Clean Architecture API
└── frontend/ # React application


## Architecture
This project follows Clean Architecture principles with separation of concerns between Domain, Application, Infrastructure, and Web API layers.
